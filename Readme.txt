To run This Project
1. Clone this repo using git clone "https://github.com/safwan72/ptsbackend.git"
2. create a virtual environment using "python -m virtualenv env", then activate using  "myvenv\Scripts\activate" 
3. Run "pip install requirements.txt"
4. Run "py manage.py runserver" to make this app running
5. Visit "http://127.0.0.1:8000/admin/", Login to admin panel using credentials
